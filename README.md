# MCP ChatBot

A CLI chatbot powered by Claude (Anthropic) using the Model Context Protocol (MCP).
This project is part of my learning through Anthropic's official MCP course.

## What it does
- Connects to Claude via Anthropic's API
- Uses an MCP server to expose document tools (read, edit)
- Lets you chat with Claude through a terminal interface

## Status
Currently working through the course TODOs in mcp_client.py and mcp_server.py.

## Tech
Python, Anthropic API, MCP Python SDK (FastMCP), async/await, prompt-toolkit
